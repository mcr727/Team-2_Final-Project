Personality Prediction Project

An assessment of an individual's personality type based on:

1. Meyer-Briggs type indicator

2. Social media posts

In Other Words:

When we analyze a large dataset of social media posts and we know the Meyer Brigg's Type Indiator (or MBTI for short) of the person behind each post, and we now take a social media post of someone whose MBTI we don't know, how accurately can we predict their MBTI?

Background Information:

The Myers Briggs Type Indicator (or MBTI for short) is a personality type system that divides everyone into 16 distinct personality types that are combinations of the following characteristics:

  1. Introversion (I) - Extroversion (E)
  2. Intuition (N) - Sensing (S)
  3. Thinking (T) - Feeling (F)
  4. Judging(J) - Perceiving(P)

Visualizations

1. Frequency of BM Types

How many of each of the 16 types are represented in the dataset

2. Word Count

3. Machine Learning

Correlations between social media post attributes and personality type
